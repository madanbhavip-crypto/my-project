# AI Virtual Painter 🎨

An AI-powered virtual drawing application built using OpenCV and MediaPipe.

## Features

- Draw in the air using your index finger
- Gesture-based color selection
- Eraser mode
- Clear canvas with open palm gesture
- Real-time hand tracking

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

pip install -r requirements.txt

## Run

python virtual_painter.py

## Controls

- ☝️ Index Finger: Draw
- ✌️ Index + Middle Finger: Select Color
- ✋ Open Palm: Clear Canvas
